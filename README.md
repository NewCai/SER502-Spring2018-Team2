## Instruction for install and run

### Install

1. Give permission to shell script:  
	chmod +x install.sh  

2. Run script:  
	./install.sh  

### Run

1. Give permission to shell script:  
	chmod +x gf.sh  

2. Compile source file  
	./gf.sh -c /path/to/source/file.gf

3. Run code  
	./gf.sh -r /path/to/inter/code.inter
